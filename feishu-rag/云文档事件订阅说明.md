# 云文档事件订阅 - 实时同步说明

多维表格或文档有变更时，飞书会推送事件到你的服务，RAG 立即同步，无需等待轮询。

## 一、在飞书开放平台添加事件

1. 进入应用 → **事件订阅**
2. Request URL 与消息事件相同，如 `https://你的ngrok地址/feishu/event`
3. **Encrypt Key**：若启用，必须在 .env 中配置 `FEISHU_ENCRYPT_KEY` 为相同值，否则 challenge 无法返回、URL 校验失败
4. 在**事件列表**中新增：
   - `drive.file.bitable_record_changed_v1`（多维表格增删改记录）
   - `drive.file.edit_v1`（文档编辑，可选）
4. 保存

## 二、权限要求

- 应用需为文档**拥有者**或**管理员**才能订阅
- 若订阅失败，请将多维表格/文档的「管理权限」授予应用或应用所在空间

## 三、启动行为

运行 `python run_bot.py` 时：

1. 首次全量同步文档
2. 自动调用 `POST /drive/v1/files/{file_token}/subscribe` 订阅配置的每个多维表格/文档
3. 之后飞书在文档变更时推送事件，服务收到后立即触发同步

## 四、禁用事件订阅

- 环境变量：`FEISHU_DRIVE_SUBSCRIBE=0`
- 或启动参数：`python run_bot.py --no-drive-subscribe`

禁用后仍使用定时轮询（默认 5 分钟）同步。
